# Food API Frontend

This is the frontend application for the Nutrition API Portal using **Next.js 14 App Router**, **TypeScript**, and **Tailwind CSS**.

## Features

- Modular architecture
- Google login (mock)
- Protected dashboard and admin pages
- API key management interface

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env.local` file based on the example below:

```env
NEXT_PUBLIC_API_BASE_URL=https://your-api-url.com
```
